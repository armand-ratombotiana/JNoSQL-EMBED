# Contributing to JNoSQL-EMBED

Thank you for considering contributing.

## Ways to Contribute

-   Reporting bugs
-   Improving documentation
-   Writing tests
-   Optimizing performance
-   Implementing storage engines

## Setup

Requirements:

-   Java 21+
-   Maven
-   Git

Clone:

git clone https://github.com/your-org/jnosql-embed.git cd jnosql-embed

Build:

mvn clean install

Run tests:

mvn test

## Pull Requests

1.  Fork repository
2.  Create feature branch
3.  Add tests
4.  Submit PR

Example:

git checkout -b feature/new-index

## Code Style

-   Follow standard Java conventions
-   Write clear commits
-   Keep PRs focused
